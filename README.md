# spatialDataExtra

Expands the data provided with the R package [`spatialData`](https://github.com/BlasBenito/spatialData)

## Datasets

### vi.gpkg
- Format: [geopackage](https://www.geopackage.org/)
- Description: larger version (30k rows) of the sf dataframe `spatialData::vi` #TODO: add link to reference page.

### quercus_env.tif
- Format: [geotiff](https://www.ogc.org/standards/geotiff/)
- Description: multilayer geotif at ~20km resolution, companion of the dataset `spatialData::quercus` #TODO: add link to reference page.
